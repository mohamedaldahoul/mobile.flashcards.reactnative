### Install 
# clone the repo 
cd to mobile.flashcards.reactnative
run ```yarn``
then run ```yarn start```

You have to have setup the emulator for Android and IOS or you have expo on Android and ISO devices.

to run
# IOS => ```yarn ios```
# android =>  ```yarn android```
